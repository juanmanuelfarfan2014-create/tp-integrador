pwd: para mostrar la carpeta actual
ls: para mostrar la lista de los archivos y carpetas 
cd: para cambiar de carpeta
mkdir: para crear una nueva carpeta
touch: para crear un archivo vacio
cat: para mostrar el contenido de un archivo en la terminal
which: para mostrar la ubicacion del ejecutable de un comando
rm: para eliminar un archivo de prueba
mv: para renombrar un archivo mal escrito
code. : para empezar abrir el directorio actual de vsc
